      32             :  * \brief Fluid disk orbiting a Kerr black hole
      33             :  *
      34             :  * The Fishbone-Moncrief solution to the 3D relativistic Euler system,
      35             :  * representing the isentropic flow of a thick fluid disk orbiting a Kerr black
      36             :  * hole. In Boyer-Lindquist coordinates \f$(t, r, \theta, \phi)\f$, the flow is
      37             :  * assumed to be purely toroidal,
      38             :  *
      39             :  * \f{align*}
      40             :  * u^\mu = (u^t, 0, 0, u^\phi),
      41             :  * \f}
      42             :  *
      43             :  * where \f$u^\mu\f$ is the 4-velocity. Then, all the fluid quantities are
      44             :  * assumed to share the same symmetries as those of the background spacetime,
      45             :  * namely they are stationary (independent of \f$t\f$), and axially symmetric
      46             :  * (independent of \f$\phi\f$). Self-gravity is neglected, so that the fluid
      47             :  * variables are determined as functions of the metric. Following the treatment
      48             :  * by Kozlowski et al. (1978) (but using signature +2) the solution is
      49             :  * expresssed in terms of the quantities
      50             :  *
      51             :  * \f{align*}
      52             :  * \Omega &= \dfrac{u^\phi}{u^t},\\
      53             :  * W &= W_\text{in} - \int_{p_\text{in}}^p\frac{dp}{e + p},
      54             :  * \f}
      55             :  *
      56             :  * where \f$\Omega\f$ is the angular velocity, \f$p\f$ is the fluid pressure,
      57             :  * \f$e\f$ is the energy density, and \f$W\f$ is an auxiliary quantity
      58             :  * interpreted in the Newtonian limit as the total (gravitational + centrifugal)
      59             :  * potential. \f$W_\text{in}\f$ and \f$p_\text{in}\f$ are the potential and
      60             :  * the pressure at the radius of the inner edge, i.e. the closest edge to the
      61             :  * black hole. Here we assume \f$p_\text{in} = 0.\f$ The solution to the Euler
      62             :  * equation is then
      63             :  *
      64             :  * \f{align*}
      65             :  * (u^t)^2 &= \frac{A}{2\Delta\Sigma}\left(1 +
      66             :  * \sqrt{1 + \frac{4l^2\Delta \Sigma^2}{A^2\sin^2\theta}}\right)\\
      67             :  * \Omega &= \frac{\Sigma}{A (u^t)^2}\frac{l}{\sin^2\theta} + \frac{2Mra}{A}\\
      68             :  * u^\phi &= \Omega u^t\\
      69             :  * W &= l\Omega - \ln u^t,
      70             :  * \f}
      71             :  *
      72             :  * where
      73             :  *
      74             :  * \f{align*}
      75             :  * \Sigma = r^2 + a^2\cos^2\theta\qquad
      76             :  * \Delta = r^2 - 2Mr + a^2\qquad
      77             :  * A = (r^2 + a^2)^2 - \Delta a^2 \sin^2\theta
      78             :  * \f}
      79             :  *
      80             :  * and \f$l = u_\phi u^t\f$ is the so-called angular momentum per unit
      81             :  * intertial mass, which is a parameter defining an
      82             :  * individual disk. In deriving the solution, an integration constant has been
      83             :  * chosen so that \f$ W\longrightarrow 0\f$ as \f$r\longrightarrow \infty\f$,
      84             :  * in accordance with the Newtonian limit. Note that, from its definition,
      85             :  * equipotential contours coincide with isobaric contours. Physically, the
      86             :  * matter can fill each of the closed surfaces \f$W = \text{const}\f$, giving
      87             :  * rise to an orbiting thick disk. For \f$W > 0\f$, all equipotentials are open,
      88             :  * whereas for \f$W < 0\f$, some of them will be closed. Should a disk exist,
      89             :  * the pressure reaches a maximum value on the equator at a coordinate radius
      90             :  * \f$r_\text{max}\f$ that is related to the angular momentum per unit inertial
      91             :  * mass via
      92             :  *
      93             :  * \f{align*}
      94             :  * l = \dfrac{M^{1/2}(r_\text{max}^{3/2} + aM^{1/2})(a^2 - 2aM^{1/2}
      95             :  * r_\text{max}^{1/2} + r_\text{max}^2)}{2aM^{1/2}r_\text{max}^{3/2} +
      96             :  * (r_\text{max} - 3M)r_\text{max}^2}.
      97             :  * \f}
      98             :  *
      99             :  * Once \f$W\f$ is determined, an equation of state is required in order to
     100             :  * obtain the thermodynamic variables. If the flow is isentropic, the specific
     101             :  * enthalpy can readily be obtained from the first and second laws of
     102             :  * thermodynamics: one has
     103             :  *
     104             :  * \f{align*}
     105             :  * \frac{dp}{e + p} = \frac{dh}{h}
     106             :  * \f}
     107             :  *
     108             :  * so that
     109             :  *
     110             :  * \f{align*}
     111             :  * h = h_\text{in}\exp(W_\text{in} - W),
     112             :  * \f}
     113             :  *
     114             :  * and the pressure can be obtained from a thermodynamic relation of the form
     115             :  * \f$h = h(p)\f$. Here we assume a polytropic relation
     116             :  *
     117             :  * \f{align*}
     118             :  * p = K\rho^\gamma.
     119             :  * \f}
     120             :  *
     121             :  * Once all the variables are known in Boyer-Lindquist coordinates, it is
     122             :  * straightforward to write them in Kerr-Schild coordinates. The coordinate
     123             :  * transformation
     124             :  *
     125             :  * \f{align*}
     126             :  * t_\text{KS} &= t\\
     127             :  * x &= \sqrt{r^2 + a^2}\sin\theta\cos\phi\\
     128             :  * y &= \sqrt{r^2 + a^2}\sin\theta\sin\phi\\
     129             :  * z &= r\cos\theta
     130             :  * \f}
     131             :  *
     132             :  * helps read the Jacobian matrix, which, applied to the azimuthal flow of the
     133             :  * disk, gives
     134             :  *
     135             :  * \f{align*}
     136             :  * u_\text{KS}^\mu = u^t(1, -y\Omega, x\Omega, 0),
     137             :  * \f}
     138             :  *
     139             :  * where \f$u^t\f$ and \f$\Omega\f$ are now understood as functions of the
     140             :  * Kerr-Schild coordinates, for which the relations
     141             :  *
     142             :  * \f{align*}
     143             :  * r^2 &= \frac{1}{2}\left(x^2 + y^2 + z^2 - a^2 +
     144             :  * \sqrt{(x^2 + y^2 + z^2 - a^2)^2 + 4a^2z^2}\right)\\
     145             :  * \theta &= \cos^{-1}\left(\frac{z}{r}\right)
     146             :  * \f}
     147             :  *
     148             :  * are needed. Finally, the spatial velocity can be readily obtained from its
     149             :  * definition,
     150             :  *
     151             :  * \f{align*}
     152             :  * \alpha v^i = \frac{u^i}{u^t} + \beta^i,
     153             :  * \f}
     154             :  *
     155             :  * where \f$\alpha\f$ and \f$\beta^i\f$ are the lapse and the shift,
